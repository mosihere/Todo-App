# Todo Web-App

You can check this Todo web-app [here](http://mosmorris.pythonanywhere.com)
<br>

## Features

1. Login
2. Logout
3. Register
4. Create Todo
5. Delete Todo
6. Edit Todo
7. Get Todo List
8. Get Todo Detail
9. Set Image For Todo
10. Update Todo Status
11. Reset Password using Email
12. Edit Profile --> First Name & Last Name & Email
